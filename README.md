# Key mappings from us to dk

| American combination | Result on danish keyboard |
|----------------------|---------------------------|
| = + RSHIFT           | \|                        |
| 2 + RALT             | @                         |
| ralt + ]             | ~                         |
| non ud backslash     | <                         |
| shift + nonusbksl    | >                         |
