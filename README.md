This repository is for a danish zmk keyboard layout for the aurora corne. 

<img src="/Picures/Keyboard_layout.png" alt="Keyboard layout"/>

| American combination | Result on danish keyboard |
|----------------------|---------------------------|
| = + RSHIFT           | \|                        |
| 2 + RALT             | @                         |
| ralt + ]             | ~                         |
| non ud backslash     | <                         |
| shift + nonusbksl    | >                         |
